Welcome to numpy-datasets' documentation!
=========================================

.. raw:: html

  <p style="text-align:justify"><a href="https://github.com/RandallBalestriero/numpy-datasets">numpy-datasets</a> is a rich in features but intuitive in use Python library that allows downloading, loading, processing and batching of various datasets from computer vision to time-series. All the functionalities only require basic Python libraries such as <a href="https://numpy.org/">Numpy</a>.</p>

.. raw:: html

  <p style="text-align:justify;font-style:italic">This is an under development research project, not an official product, expect bugs and sharp edges; please help by trying it out, reporting bugs and missing pieces.</p>
